# PHP 语法

``` php
<?php
// 常量名
define("GREETING", "欢迎访问 Runoob.com");
echo GREETING;    // 输出 "欢迎访问 Runoob.com"

// 并置运算符 (.) 用于把两个字符串值连接起来
$txt1="Hello world!";
$txt2="What a nice day!";
echo $txt1 . " " . $txt2;

// 数组
$cars=array("Volvo","BMW","Toyota");
echo "I like " . $cars[0] . ", " . $cars[1] . " and " . $cars[2] . ".";
?>
```
